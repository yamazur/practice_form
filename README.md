# Тест-кейс 1: Отображение формы регистрации

Приоритет: Высокий
Предусловие: Пользователь имеет доступ к сайту

Шаги:

1. Открыть страницу регистрации https://demoqa.com/automation-practice-form
2. Убедиться, что URL и заголовок страницы корректны
3. Проверить наличие всех элементов формы:

- Поля ввода: First Name, Last Name, Email, Mobile Number, Date of Birth, Current Address
- Radio buttons: Gender (Male, Female, Other)
- Checkboxes: Hobbies (Sports, Reading, Music)
- Выпадающие списки: Subjects, State, City
- Кнопка загрузки файла: Picture
- Кнопка отправки: Submit

Ожидаемый результат:

1. Страница загружена, URL корректный
2. Все элементы формы отображаются

# Тест-кейс 2: Успешная регистрация с валидными данными (Positive Test)

Шаги:

1. Заполнить поле First Name: "Ivan"
2. Заполнить поле Last Name: "Pupkin"
3. Заполнить поле Email: "ivan.pupkin@testqa.com"
4. Выбрать Gender: "Male" (клик на radio button)
5. Заполнить поле Mobile: "8900123456"
6. Выбрать Date of Birth через datepicker: 15 May 1995
7. В поле Subjects ввести "English", нажать Enter
8. Добавить второй subject: "Maths"
9. Выбрать Hobbies: Sports (чекбокс) и Music (чекбокс)
10. Загрузить Picture: выбрать файл "test_photo.jpg"
11. Заполнить Current Address: "123 Test Street, Springfield, IL 62701"
12. Выбрать State из выпадающего списка: "NCR"
13. После выбора State выбрать City: "Delhi"
14. Нажать кнопку Submit
15. Закрыть модальное окно подтверждения (если появится)

Ожидаемый результат:

1. Все поля заполнены без ошибок валидации
2. После нажатия Submit появляется модальное окно с заголовком "Thanks for submitting the form"
3. В модальном окне отображаются все введенные данные корректно
